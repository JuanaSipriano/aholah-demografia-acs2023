# Taholah-demografia-acs2023
Análisis demográfico de Taholah con ACS 2023

Este proyecto realiza un análisis exploratorio de datos (EDA) de la comunidad de **Taholah** utilizando datos del portal Census Reporter 2023 . El objetivo principal fue **filtrar, agrupar y transformar los datos para generar una nueva tabla resumida**, y a partir de ella construir visualizaciones que facilitan la interpretación de los principales indicadores demográficos.
___
## Acciones 

- Filtrar las variables relevantes del ACS para Taholah.
- Agrupar datos por categorías (edad, etnia, movilidad, hogares, ingresos).
- Calcular porcentajes con respecto al total poblacional.
- Crear una tabla final consolidada para análisis y visualización.
- Generar gráficos para interpretar patrones demográficos clave.
